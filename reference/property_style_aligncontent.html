alignContent style 속성

alignContent 속성은 항목이 교차 축 (세로)에서 사용 가능한 공간을 모두 사용하지 않는 경우 유연한 컨테이너의 항목을 정렬합니다.

참고 : justifyContent 속성을 사용하면 항목을 주축 (가로)에 맞춥니다.

* 이 속성이 적용되려면 여러 줄의 항목이 있어야 합니다.

coral
lightblue
khaki
pink
lightgrey
lightgreen

버튼을 클릭하여 alignContent 속성 값을 "space-between"으로 설정합니다.

구문(Syntax)

object.style.alingnContent

alignContent 속성 설정

align-content 속성은 flex-wrap 속성의 동작을 수정한다.
align-items 와 유사 하지만 플렉스 항목을 정렬하는 대신 플렉스 라인을 정렬한다.
참고 : 이 속성이 적용되려면 여러 줄의 항목이 있어야 한다!
참고 : justify-content 속성은 항목을 주축 (가로)에 정렬한다.

  div {
    width: 70px;
    height: 300px;
    border: 1px solid #ccc;
    display: flex;
    flex-wrap: wrap;
    align-content: center;
  }
object.style.alignmentContent = "stretch|center|flex-start|flex-end|space-between|space-around|initial|inherit"

속성값

속성값설명
stretch기본값, 컨테이너에 맞게 늘어난다.
center중앙에 배치된다.
flex-start시작 부분에 품목이 배치된다.
flex-end 끝에 배치된다.
space-between공백으로 균등 배치된다
space-around앞, 뒤, 중간에 공백으로 배치된다.
initial기본값(stretch)으로 설정한다.
inherit상위 요소에서 속성을 상속한다.
