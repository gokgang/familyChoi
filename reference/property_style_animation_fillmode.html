JavaScript Style animationFillMode 속성

animationFillMode 속성은 애니메이션이 재생되지 않을 때 (완료 될 때 또는 "지연"이 있을 때) 요소에 적용할 스타일을 지정한다.
기본적으로 CSS 애니메이션은 첫 번째 키 프레임이 "재생"될 때까지 애니메이션 요소에 영향을주지 않으며 마지막 키 프레임이 완료되면 영향을 멈춘 다. animationFillMode 속성은 이 동작을 재정의할 수 있다.

animationFillMode 속성 변경

document.getElementById("myDivision").style.animationFillMode = "forwards";
Animation

애니메이션이 끝났을 때 마지막 키 프레임에서 설정한 스타일을 {left : 200px;}로 유지하려면 버튼 클릭.

구문(Syntax)

animationFillMode 속성 반환

object.style.animationFillMode;

animationFillMode 속성 설정

object.style.animationFillMode = "none|forwards|backwards|both|initial|inherit";

속성 값

속성 값설 명
none기본값. 애니메이션 실행 전이나 실행 후 대상에 스타일을 적용하지 않습니다.
forwards 애니메이션 종료 후 종료된 시간의 keyframes 속성 값을 그대로 적용
backwards 애니메이션 종료 후 애니메이션의 첫 번째 반복을 시작할 keyframes에 정의된 속성 값을 적용
both 애니메이션은 forwards 와 backwards 모두 규칙을 따릅니다. 즉, 애니메이션 속성을 양방향으로 확장한다.
initial기본값으로 설정
inherit상위 요소에서 상속
