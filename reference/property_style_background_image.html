JavaScript Style backgroundImage 속성

backgroundImage 속성은 요소의 배경 이미지를 설정하거나 반환한다.
참고 : 배경 이미지 외에 배경색도 함께 지정해야 한다.
이미지를 사용할 수 없는 경우 배경색이 사용된다.

문서의 배경 이미지 설정

document.body.style.backgroundImage = "url('img_tree.png')";

구문(Syntax)

backgroundImage 속성 반환

object.style.backgroundImage

backgroundImage 속성 설정

object.style.backgroundImage = "url('URL')|none|initial|inherit"

속성 값

속성 값설 명
url('URL')이미지 파일의 위치
none배경 이미지 없음. 기본값
initial기본값으로 설정
inherit상위 요소에서 상속
