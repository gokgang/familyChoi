JavaScript Style backgroundPosition 속성

backgroundPosition 속성은 요소 내 배경 이미지의 위치를 설정하거나 반환한다.

배경 이미지의 위치 설정

document.body.style.backgroundPosition = "top right";

구문(Syntax)

backgroundPosition 속성 반환

object.style.backgroundPosition

backgroundPosition 속성 설정

object.style.backgroundPosition = "value"

속성 값

속성 값설 명
top left
top center
top right
center left
center center
center right
bottom left
bottom center
bottom right
 키워드를 하나만 지정하면 다른 값은 "center" 이다.
x% y% x 값은 가로 위치를 나타내고, y 값은 세로 위치를 나타낸다.
왼쪽 위 모서리는 0% 0%이다.
오른쪽 아래 모서리는 100% 100%이다.
하나의 값만 지정하면 다른 값은 50%이다.
xpos ypos x 값은 가로 위치를 나타내고, y 값은 세로 위치를 나타낸다.
왼쪽 위 모서리는 0 0이다.
단위는 픽셀(0px 0px) 또는 다른 CSS 단위일 수 있다. 하나의 값만 지정하면 다른 값은 50%이다.
%와 단위를 혼합할 수 있다.
initial기본값으로 설정
inherit상위 요소에서 상속

예제

배경 이미지 위치를 가운데 아래로 변경

document.getElementById("id").style.backgroundPosition = "center bottom";

배경 이미지 위치를 가로 200 픽셀 , 세로 40 픽셀로 변경

document.getElementById("id").style.backgroundPosition = "200px 40px";

배경 이미지 위치 반환

document.getElementById("id").style.backgroundPosition;
