JavaScript Style animationDirection 속성

animationDirection 속성은 애니메이션이 대체주기에서 역방향으로 재생되어야 하는지 여부를 설정하거나 반환한다. 애니메이션이 한 번만 재생되도록 설정된 경우에는 효과가 없다.

animationDirection 속성 변경

document.getElementById("myDivision").style.animationDirection = "reverse";
Animation

구문(Syntax)

animationDirection 속성 반환

let direction = object.style.animationDirection;

animationDirection 속성 설정

object.style.animationDirection = "normal|reverse|alternate|alternate-reverse|initial|inherit";

속성 값

속성 값설 명
normal기본값. 정상적으로 재생
reverse역방향으로 재생
alternate홀수 (1,3,5 등)에 정상 재생, 짝수 (2,4,6 등...)에 역방향 재생
alternate-reverse홀수(1,3,5 등)에 역방향 재생, 짝수(2,4,6 등...)에 정상 방향 재생
initial기본값으로 설정
inherit상위 요소에서 상속
