JavaScript Style animationDelay 속성

animationDelay: 5s

animationDelay 속성 변경

document.getElementById("myDIV").style.animationDelay = "2s";

animationDelay 속성은 애니메이션이 시작되는시기를 지정한다.
animationDelay 값은 초 (s) 또는 밀리 초 (ms)로 지정한다.
참고 : 음수 값이 허용된다. -2s는 애니메이션을 한 번에 시작하지만 애니메이션은 2 초 후에시작한다.
지연 시간은 버튼을 클릭 할 때가 아니라 페이지로드가 완료된 때부터 계산된다.
따라서 이미 애니메이션이 동작하고 있는 데 지연시간을 변경하는 것은 의미가 없다.

구문(Syntax)

animationDelay 속성 반환

const delay = object.style.animationDelay;

animationDelay 속성 설정

object.style.animationDelay = "time|initial|inherit";

속성 값

속성 값설 명
time선택 사항, 시작 대기 시간(초 또는 밀리초) 지정. 기본값은 0
initial기본값으로 설정
inherit상위 요소에서 상속
