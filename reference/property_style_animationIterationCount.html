JavaScript Style animationIterationCount 속성

animationIterationCount 속성은 애니메이션의 재생 횟수를 설정하거나 반환한다.

myAanimation

animationIterationCount 속성 변경

document.getElementById("myMove").style.animationIterationCount = "infinite";

구문(Syntax)

animationIterationCount 속성 반환

const count = object.style.animationIterationCount;

animationIterationCount 속성 설정

object.style.animationIterationCount = "number|infinite|initial|inherit"

속성 값

속성 값설 명
number재생해야 하는 횟수 지정. 기본값은 1
infinite무한 반복으로 재생하도록 지정
initial기본값으로 지정
inherit상위 요소에서 상속
