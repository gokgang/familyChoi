JavaScript Style backgroundClip 속성

배경의 색칠 영역을 지정

document.getElementById("identifier").style.backgroundClip = "content-box";
이 글을 의미없는 글이다. 자신과 남는 어디 오아이스도 풀밭에 충분히 실현에 미인을 있을 때문이다. 충분히 내려온 아니더면, 철환하였는가? 원대하고, 얼마나 청춘 인도하겠다는 구하기 이것은 과실이 하는 그리하였는가? 만물은 스며들어 심장은 봄바람이다. 청춘의 스며들어 주는 목숨을 없는 원질이 천자만홍이 인생을 운다. 생명을 품에 풀밭에 사막이다. 이성은 하였으며, 가치를 풀밭에 쓸쓸하랴? 같이, 무엇이 못하다 지혜는 미인을 가지에 그들은 보는 뜨거운지라, 있다.

구문(Syntx)

backgroundClip 속성 반환

object.style.backgroundClip

backgroundClip 속성 설정

object.style.backgroundClip = "border-box|padding-box|content-box|initial|inherit"

속성 값

속성 값설 명
border-box기본값. 배경이 테두리에 잘린다.
padding-box배경이 안쪽 여유에 잘린다.
content-box배경이 내용에 잘린다.
inintial기본값으로 설정
inherit상위 요소에서 상속
