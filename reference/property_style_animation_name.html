JavaScript Style animationName 속성

document.getElementById("id").style.animationName = "myMove";

animationName 속성은 @keyframes 애니메이션의 이름을 설정하거나 반환한다.

구문(Syntax)

animationName 속성 반환

let keyframes = object.style.animationName;
: 속성 변경 후 확인

animationName 속성 설정

objectstyle.animationName = "none|keyframeName|initial|inherit";

속성 값

속성 값설 명
none기본값. 애니메이션이 없음(cascade에서 애니메이션을 재정의하는 데 사용할 수 있다)
keyframeName선택할 키 프레임의 이름 지정
initial기본값으로 설정
inherit상위 요소에서 상속
