JavaScript Style alignSelf 속성

컨테이너 내부의 선택 항목에 대한 정렬

참고 : alignSelf 속성은 컨테이너의 alignItems 속성을 재정의한다.

Example

document.getElementById("myBlueDiv").style.alignSelf = "stretch";

Syntax

alignSelf 속성 반환

object.style.alignSelf

alignSelf 속성 설정

object.style.alignSelf = "auto|stretch|center|flex-start|flex-end|baseline|initial|inherit";

속성 값

속성 값설 명
auto기본값. 상위 컨테이너의 정렬 항목 속성에서 상속. 상위 컨테이너가 없는 경우 "stretch"
stretch요소가 컨테이너에 맞도록 늘여서 배치
center중앙에 배치
flex-start맨 위 시작 부분에 배치
flex-end맨 아래 부분에 배치
baseline기준선에 배치
initial기본값으로 설정
inherit상위 요소에서 상속
align-items: flex-start
alignSelf: center
align-items: flex-start

버튼을 클릭하면 alignSelf 속성 값을 변경한다.
