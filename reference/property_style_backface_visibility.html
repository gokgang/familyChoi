JavaScript Style backfaceVisibility 속성

backfaceVisibility 속성은 회전하는 요소가 화면을 향하지 않을 때 요소를 표시할지 여부를 설정한다. 이 속성은 회전하는 요소의 뒷면을 보고 싶지 않을 때 유용하다.

회전하는 <div> 요소의 뒷면 숨기기

document.getElementById("idDiv").style.backfaceVisibility = "hidden";

backfaceVisibility

구문(Syntax)

backfaceVisibility 속성 반환

object.style.backfaceVisibility

backfaceVisibility 속성 설정

object.style.backfaceVisibility = "visible|hidden|initial|inherit";

속성 값

속성 값설 명
visible기본값, 앞 뒷면 모두 보이기
hidden뒷면 숨기기
initial기본값으로 설정
inherit상위 요소에서 상속
