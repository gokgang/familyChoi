JavaScript Style backgroundRepeat 속성

backgroundRepeat 속성은 배경 이미지를 반복(타일)하는 방법을 설정하거나 반환한다.

배경 이미지를 반복 안함으로 설정

document.body.style.backgroundRepeat = "no-repeat'";

구문(Syntax)

backgroundRepeat 속성 반환

object.style.backgroundRepeat

backgroundRepeat 속성 설정

object.style.backgroundRepeat = "repeat|repeat-x|repeat-y|no-repeat|initial|inherit"

속성 값

속성 값설 명
epeat수직 및 수평으로 반복. 기본값
repeat-x수평으로만 반복
repeat-y수직으로만 반복
no-repeat반복되지 않는다
initial기본값으로 설정
inherit상위 요소에서 상속
