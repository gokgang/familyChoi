JavaScript Style background 속성

background 속성은 약식 형식으로 최대 8 개의 개별 배경 속성을 설정하거나 반환한다. 이 속성을 사용하여 순서에 상관 없이 다음 중 하나 이상을 설정하거나 반환 할 수 있다.

  • 배경색 : background-color
  • 배경 이미지 : background-image
  • 배경 반복 : background-repeat
  • 배경 첨부 : background-attachment
  • 배경 위치 : background-position
  • 배경 크기 : background-size
  • 배경 영역 : background-origin
  • 배경 도색 : background-clip

위의 속성은 별도의 스타일 속성으로 설정할 수도 있다. 더 나은 제어 가능성을 위해 고급 작성자가 아닌 경우 별도의 속성을 사용하는 것이 좋다.

기본값 : transparent none repeat scroll 0% 0% auto padding-box border-box

반환 값 : 요소의 배경을 나타내는 문자열

문서 배경 스타일 지정

document.body.style.background = "#f3f3f3 url('img_tree.png') no-repeat right top";

구문(Syntax)

배경 속성 반환

let background = object.style.background;

배경 속성 설정

object.style.background = "color image repeat attachment position size origin clip|initial|inherit";

속성 값

속성 값설 명
color요소의 배경색 설정
image요소의 배경 이미지 설정
repeat배경 이미지를 반복하는 방법 설정
attachment배경 이미지를 고정할 지 또는 페이지와 함께 스크롤할 지 설정
position배경 이미지의 시작 위치 설정
size배경 이미지의 크기를 설정
origin배경 위치 지정 영역을 설정
clip배경 이미지의 도색 영역 설정
initial기본값으로 설정
inherit상위 요소에서 상속

배경 변경

document.getElementById("myDIV").style.background = "url('smiley.gif') blue repeat-x center";

문서의 배경색 설정

document.body.style.backgroundColor = "red";

문서의 배경 이미지 설정

document.body.style.backgroundImage = "url('img_tree.png')";

배경 이미지를 반복 안함으로 설정

document.body.style.backgroundRepeat = "repeat-y";

스크롤할 때 고정할 배경 이미지 설정

document.body.style.backgroundAttachment = "fixed";

배경 이미지의 위치 변경

document.body.style.backgroundPosition = "top right";

문서의 배경 속성 값 반환

let background = document.body.style.background;
