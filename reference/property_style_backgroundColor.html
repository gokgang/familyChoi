JavaScript Style backgroundColor 속성

backgroundColor 속성은 요소의 바탕색을 설정하거나 반환한다.

문서의 바탕색 설정

document.body.style.backgroundColor = "red";

구문(Syntax)

backgroundColor 속성 반환

object.style.backgroundColor

backgroundColor 속성 설정

object.style.backgroundColor = "color|transparent|initial|inherit"

속성 값

속성 값설 명
color바탕색 지정. 가능한 색상 값은 CSS 색상 값
transparent기본값, 바탕색 투명(내용의 바탕색)
initial기본값으로 설정
inherit상위 요소에서 상속
요소의 바탕색 설정과 반환
