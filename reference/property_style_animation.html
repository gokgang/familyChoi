JavaScript Style animation 속성

animation 속성 값

속성 값설 명
animationName 선택자에 사용할 keyframe의 이름 지정
animationDuration 완료하는 데 걸리는 시간(초 또는 밀리초) 지정
animationTimingFunction 애니메이션의 속도 곡선 지정
animationDelay 애니메이션을 시작하기 전의 지연을 지정
animationIterationCount 애니메이션을 재생할 횟수를 지정
animationDirection 대체 사이클에서 애니메이션을 반대로 재생할지 여부를 지정
animationFillMode 애니메이션이 실행 중인 시간 이외의 시간에 적용할 값 지정
animationPlayState 애니메이션이 실행 중인지 또는 일시 중지되었는지 여부 지정
initial 기본값으로 설정
inherit 상위 요소에서 상속

기본값 : none 0 실행안 함, ease 1 정상 실행

반환값 : 요소의 애니메이션 속성을 나타내는 문자열

참고 : 항상 animationDuration 속성을 지정해야 한다.
그렇지 않으면 시간이 0이 되므로 재생되지 않는다.
