JavaScript Style backgroundAttachment 속성

backgroundAttachment 속성은 문서가 스크롤될 때 배경 이미지를 고정할 지 여부를 설정하거나 설정값을 반환한다.

배경 이미지 스크롤 고정

document.body.style.backgroundAttachment = "fixed";

구문(Sytax)

backgroundAttachment 속성 반환

object.style.backgroundAttachment

backgroundAttachment 속성 설정

object.style.backgroundAttachment = "scroll|fixed|local|initial|inherit"

속성 값

속성 값설 명
scroll기본값, 배경 그림은 요소와 함께 스크롤
fixed배경 그림은 화면에 고정
local배경 그림은 요소의 내용과 함께 스크롤
initial기본값으로 설정
inherit상위 요소에서 상속
