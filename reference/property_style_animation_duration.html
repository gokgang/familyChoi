JavaScript Style animationDuration 속성

document.getElementById("myDIV").style.animationDuration = "3s";

animationDuration 속성은 애니메이션이 한주기를 완료하는 데 걸리는 시간 (초 또는 밀리 초)을 정의한다.

구문(Syntax)

animationDuration 속성 반환

object.style.animationDuration;

animationDuration 속성 설정

object.style.animationDuration = "time|initial|inherit"

속성 값

속성 값설 명
time 애니메이션을 한 주기를 완료하는 데 걸리는 시간으로 기본값은 0이며, 애니메이션 없음이다.
initial기본값으로 설정
inherit상위 요소에서 상속
