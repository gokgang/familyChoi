JavaScript Style animationPlayState 속성

animationPlayState 속성은 애니메이션 실행 중에 일시 중지하려면이 속성을 사용한다.

애니메이션 일시 중지

document.getElementById("division").style.animationPlayState = "paused";
Animation

구문(Syntax)

animationPlayState 속성 반환

let playState = object.style.animationPlayState;

animationPlayState 속성 설정

object.style.animationPlayState = "running|paused|initial|inherit";

속성 값

속성 값설 명
running기본값. 애니메이션 실행 지정
paused 애니메이션 일시 중지 지정
initial기본값으로 설정
inherit상위 요소에서 상속
