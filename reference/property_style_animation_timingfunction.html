JavaScript Style animationTimingFunction 속성

animationTimingFunction 속성 변경

document.getElementById("myMove").style.animationTimingFunction = "linear";

animationTimingFunction 은 애니메이션의 속도 곡선을 지정한다.
속도 곡선은 애니메이션이 CSS 스타일에서 다른 스타일로 변경하는 데 사용되는 시간을 지정한다.
속도 곡선은 애니메이션을 부드럽게 작동하도록 하는 데 사용된다.

구문(Syntax)

animationTimingFunction 속성 반환

let timing = object.style.animationTimingFunction;

animationTimingFunction 속성 설정

object.style.animationTimingFunction = "linear|ease|ease-in|ease-out|cubic-bezier(n, n, n, n )|initial|inherit"

속성 값

속성 값설 명
linear시작부터 끝까지 속도 동일
ease기본값, 빠르게 시작해서 느리게 끝난다.
ease-in시작 부분의 속도가 느리다.
ease-out끝 부분의 속도가 느리다.
ease-in-out시작과 끝 부분의 속도가 느리다.
cubic-bezier(n, n, n, n)입방체 함수에 고유 값 지정, 가능한 값은 0부터 1까지의 숫자
initial기본값으로 설정
inherit상위 요소에서 상속
