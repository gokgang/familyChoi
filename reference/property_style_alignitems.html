JavaScript Style alignItems 속성

<div> 요소의 모든 항목을 시작 부분에 정렬한다.

참고 : align-items 속성을 재정의하려면 각 항목의 align-self 속성을 사용해야 한다.

Syntax

alignItems 속성 반환

object.style.alignItems

alignItems 속성 설정

object.style.alignItems = "stretch|center|flex-start|flex-end|baseline|initial|inherit";

속성 값

속성 값설 명
stretch기본값. 컨테이너에 맞게 품목이 늘어난다.
center중앙에 배치
flex-start시작 부분에 품목 배치
flex-end끝 부분에 품목 배치
baseline컨테이너의 기준선에 배치
initial기본값으로 설정
inherit상위 요소에서 상속
